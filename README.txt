PrimeTime Manual REG System (Local Search)

FILES:
- ocr_extract_text.py: Extracts text from PDF using Tesseract OCR
- qa_search.py: Lets you search the extracted text using REG (regex) patterns
- primetime_manual.txt: OCR output of the uploaded PrimeTime User Guide
- README.txt: You're reading it

USAGE:
1. Run 'ocr_extract_text.py' to OCR any PDF.
2. Then run 'qa_search.py' and enter a command like 'report_timing' or 'setup time'.
